# 📈 Stock Market ETL & Analytics Dashboard

## Project Overview

The **Stock Market ETL & Analytics Dashboard** is an end-to-end Data Engineering project that automates the extraction, transformation, storage, and visualization of Indian stock market data. The project demonstrates a complete ETL pipeline by collecting daily stock market data from the BSE API, performing data cleaning and transformation in Python, storing the processed data in Microsoft SQL Server, and generating interactive business insights via a Power BI dashboard.

This project follows industry-standard ETL practices, including data extraction, profiling, validation, transformation, loading, and analytical reporting. It provides meaningful insights into stock performance, market trends, industry distribution, price movements, and key financial indicators while showcasing efficient database design and SQL query optimization.

## Tech Stack

* **Programming Language:** Python
* **Database:** Microsoft SQL Server
* **Data Processing:** Pandas, NumPy
* **Data Extraction:** BSE API (`bsedata`)
* **Database Connectivity:** SQLAlchemy, PyODBC
* **Data Visualization:** Power BI
* **Query Language:** SQL (T-SQL)
* **Development Environment:** Jupyter Notebook, Visual Studio Code
* **Version Control:** Git & GitHub

## Key Features

* Automated stock market data extraction from the BSE API
* End-to-end ETL pipeline implementation
* Data cleaning, validation, and transformation
* SQL Server database integration
* Optimized SQL queries for business analysis
* Interactive Power BI dashboard for reporting and visualization

## ETL Workflow

**Extract → Transform → Load → Analyze → Visualize**

* **Extract:** Collects daily stock market data from the BSE API and CSV files.
* **Transform:** Cleans, validates, standardizes, and prepares the data for analysis.
* **Load:** Stores the transformed data into Microsoft SQL Server.
* **Analyze:** Executes SQL queries to generate business insights and KPIs.
* **Visualize:** Presents interactive reports and dashboards using Power BI.
